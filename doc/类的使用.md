    }

# 公共，私有与受保护的修饰符
> 默认是`public`

- `public` 公共
- `private` 私有
- `protected` 可以在派生类中使用

示例：
```ts
class Person{
    public name:string;
    private money:number;
    protected sex:'男' | '女';
}
```
